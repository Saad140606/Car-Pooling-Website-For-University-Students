## 🎓 REUSABLE PATTERNS

### Pattern 1: Animated List
```tsx
<PageTransition>
  <StaggeredContainer>
    {items.map((item, idx) => (
      <Reveal key={item.id} delay={idx * 80}>
        <AnimatedCard>{item}</AnimatedCard>
      </Reveal>
    ))}
  </StaggeredContainer>
</PageTransition>
```

### Pattern 2: Form with Validation
```tsx
<FormField
  label="Field Name"
  value={value}
  onChange={setValue}
  error={validationError}
  success={isValid && value}
  icon={<Icon />}
/>
```

### Pattern 3: Data Table
```tsx
<DataGrid
  columns={columns}
  data={data}
  pageSize={10}
  selectable
  onSelectionChange={handleSelect}
/>
```

### Pattern 4: Status Flow
```tsx
<StepIndicator
  steps={steps}
  currentStep={activeStep}
  onStepChange={setActiveStep}
/>
```

### Pattern 5: Feedback Modal
```tsx
<AnimatedModal
  type="success"
  title="Success!"
  actions={[{ label: 'Done', onClick: close }]}
/>
```
